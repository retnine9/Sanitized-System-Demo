# Sanitized-System-Demo
A sanitized system demonstration reflecting the end to end trading system. All logic/methodologies I consider to be proprietary (such as feature engineering, data sources, actual model weights, etc) are not included/not at their fullest extent, so this is mostly an architectural demonstration. The entire system would allow you to lightweight train a model and attach it to the trading system and deploy it live, assuming you have a polymarket account to attach it to. Performance will absolutely not be good, given I will not provide the full methodology of how I train the models.
